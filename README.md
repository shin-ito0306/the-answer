# The Answer

## 概要
質問投稿、回答投稿サイトです。
分からない事、困っている事、悩み事などなんでも質問することが出来ます。
また、他の人の質問を回答し、質問者からベストアンサーに選ばれると、報酬として
質問者からポイントを獲得することが出来ます。
